About this Website

This website showcases my own Photon based project. The project consists of a basement leak monitoring system, that will notify me if any leaks occur in a specific area of my basement. It uses push notifications sent using Pushover service. A secondary use is to send data from a connected DHT22 (temperature/humidity) sensor to a Django written database application, for collection and analysis.

The Photon

A Micro-controller created by Particle (formerly known as Spark). It is Wi-Fi capable, and this means that your ideas can connect to the Internet. The programming language (based on C/C++) is very easy to learn and use. Everything you need to know about how to use the Photon can be found on the Particle.io website.
